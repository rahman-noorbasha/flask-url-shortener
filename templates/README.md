# Flask URL Shortener

A web-based URL shortener built using Flask.

## Features

- Shorten long URLs
- Generate unique short codes
- Redirect shortened links
- Browser-based interface
- Clean responsive UI

## Technologies Used

- Python
- Flask
- HTML
- CSS

## Run Locally

```bash
pip install flask
python app.py
```

Open:

http://127.0.0.1:5000

## Learning Outcomes

This project demonstrates:

- Flask routing
- Form handling
- URL redirects
- Web application architecture
- Backend request-response cycle
